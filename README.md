# Vichaar

> Think. Reflect. Learn.

AI-powered learning platform that tests your understanding through adaptive questioning — not multiple choice, but real thinking.

---

## Stack

**Frontend** — Next.js, Tailwind CSS v4, NextAuth  
**Backend** — Node.js, Express, MongoDB  
**AI Model** — FastAPI, LangChain, Google Gemini, ChromaDB

---

## Project Structure

```
aiLearner/
├── client/          # Next.js frontend
├── server/          # Express backend
└── RAG-based-QnA/   # Python AI model
```

---

## Features

- Google OAuth authentication
- Adaptive AI questioning based on your answers
- Per-answer evaluation — factuality, context, originality, examples
- Session analysis with strengths, improvements and next steps
- Full learning history
